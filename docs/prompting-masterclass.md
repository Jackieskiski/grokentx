# LTX-2.3 Prompting Masterclass

**Goal:** Consistent, high-quality, cinematic or erotic video with strong motion, character consistency, anatomy/physics, and audio sync.

LTX-2.3 responds extremely well to **long, flowing, single-paragraph prompts** in present tense.

## Core Prompt Formula

1. **Camera / Motion first** (most important)
2. **Subject + one primary action**
3. **Full scene details** (lighting, environment, textures)
4. **Cinematic references**
5. **Audio elements** (very important for LTX)
6. **LoRAs & weights at the very end**

## Key Techniques

- Use present tense + active verbs
- Be specific with micro-details and physics
- For NSFW: direct but natural language + right LoRAs
- Strong reference image + IC-LoRA for character consistency
- Test short → pick best → extend

## Workflow Recommendation

Best starting point: Image-to-Video (`animate-ltx23`) with a strong reference image, then extend.
