# grokentx

LTX-2.3 workflows, prompts, LoRA stacks, and tools.

Public repo for organizing and iterating on LTX-2.3 video generation work (Pirate Diffusion / Graydient + local ComfyUI).

## Current Structure

- `docs/` — Prompting guides, troubleshooting, parameter cheatsheets
- `prompts/` — Tested prompts (cinematic + NSFW)
- `lora-library/` — LoRA triggers, weights, and proven stacks
- `workflows/` — ComfyUI and Pirate Diffusion workflow notes
- `scripts/` — Helper tools
- `assets/` — Reference images (when added)
